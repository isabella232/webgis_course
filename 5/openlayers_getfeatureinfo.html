Домой О курсе Любой сервер WMS в обязательном порядке поддерживает два основных запроса: * **GetCapabilities** * **GetMap** Остальные запросы, описанные в стандарте, поддерживаются факультативно. Среди них есть и GetFeatureInfo, позволяющий запрашивать атрибутивную информацию об объекте, расположенном по указанным координатам. Именно этот тип запроса используется во многих клиентских приложениях и со стороны пользователя может выглядеть, например, как всплывающее окно с информацией об объекте, появляющееся после щелчка на объекте. Добавим на карту контрол **WMSGetFeatureInfo**, который будет выполнять запросы GetFeatureInfo: <html> <head> <title>My Map</title> <script src="http://openlayers.org/dev/OpenLayers.js"></script> <script> function init() { var options = { div: "mapid", projection: "EPSG:3857" }; var map = new OpenLayers.Map(options); var osm = new OpenLayers.Layer.OSM() var imagery = new OpenLayers.Layer.WMS( "Sattelite Image of St. Petersburg", "http://10.22.0.9/cgi-bin/wms", {layers: "railway_station", transparent: true}, {isBaseLayer: false, opacity: 0.9} ); var extent = [3322236.017526,8366084.2704585,3418852.421265,8395053.9041745]; map.addLayers([osm, imagery]); map.zoomToExtent(extent); var click_ctrl = new OpenLayers.Control.WMSGetFeatureInfo({ url: "http://10.22.0.9/cgi-bin/wms", layers: [imagery], queryVisible: true, infoFormat: 'application/vnd.ogc.gml', maxFeatures: 50, autoActivate: true, // Vendor-specific параметр, используемый в MapServer (радиус поиска объектов в пикселях) vendorParams: { radius: 10 }, eventListeners: { 'getfeatureinfo': function(e) { // Текстовое представление ответа сервера console.log(e.text); // Координаты клика console.log(e.xy); // Список объектов OpenLayers.Feature.Vector, содержащих в атрибутике информацию, полученную с сервера console.log(e.features); } } }); // Добавляем контрол на карту map.addControl(click_ctrl); } </script> </head> <body> <body onload="init()"> <div id="mapid" style="width:100%; height:100%"></div> </body> </html> Для запроса данных c помощью данного контрола необходимо, чтобы приложение и WMS-сервис функционировали в рамках одного домена, поэтому разместите файл с картой на Web-сервере и сделайте на него симлинк в `/var/www/`. Запустите [Firebug](http://firebug.ru/) и откройте вкладку *Консоль*, после этого кликните на каком-либо объекте слоя *imagery* (в данном случае это слой железнодорожных станций), после чего OpenLayers в фоне выполнит запрос GetFeatureInfo, результат которого будет выведен в консоль Firebug.
