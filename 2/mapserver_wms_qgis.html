Чтобы убедиться в том, что наши данные опубликованы корректно, попробуем визуализировать их в настольном клиенте QGIS. Так как QGIS - кроссплатформенное приложение, то для примера будем использовать QGIS, запущенный в Windows 7. 1. Поскольку WMS-сервер отрисовывает данные в соответствии с переданным охватом, то выполним небольшие предварительные действия: * Подключимся к WFS-серверу `http://10.22.0.9/cgi-bin/wfs` согласно [инструкции](mapserver_wfs_qgis.html) и откроем слой железнодорожных станций; * Включим перепроецирование на лету в проекцию EPSG:3857: `Установки -> Свойства проекта -> Система координат -> Включить преобразование координат "на лету"`: ![][02-wms-qgis-01] 2. В меню *Слой* выберите пункт *Добавить WMS-слой*; 3. Нажмите кнопку *Создать* и введите следующие параметры: * **Имя** - произвольная строка по которой вы сможете идентифицировать данное WMS-соединение в списке доступных соединений; * **URL** - адрес WMS-сервера; 4. В открывшемся окне нажмите кнопку *Подключиться*, после чего вы увидите список доступных WMS-слоёв, опубликованных на сервере: ![][02-wms-qgis-03] 5. Выберите один из слоёв (убедитесь, что слой запрашивается в проекции *WGS 84 / Pseudo Mercator*) и нажмите кнопку *Добавить*. Отключите WFS-слой железнодорожных станций; 6.
