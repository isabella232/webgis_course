Домой О курсе ![][02-mapserver-01] Установка веб-сервера Apache ---------------------------- Для установки веб-сервера Apache выполняем следующую команду (на том же хосте, где запущен PostgreSQL): sudo aptitude install apache2 Чтобы убедиться в том, что Apache установился корректно, перейдите в браузере по адресу `http://IP-адрес веб-сервера`. Дожна отобразиться страница, содержащая следующий текст: It works! This is the default web page for this server. The web server software is running but no content has been added, yet Установка MapServer ------------------- Для установки MapServer выполним следующую команду: sudo aptitude install cgi-mapserver mapserver-bin Чтобы убедиться в том, что MapServer установился корректно, перейдите в браузере по адресу `http://IP-адрес веб-сервера/cgi-bin/mapserv`. Должна отобразиться страница, содержащая следующий текст: No query information to decode. QUERY_STRING is set, but empty. Если страница с таким содержимым не появилась, попробуйте включить модуль *cgid*: sudo a2enmod cgid sudo service apache2 restart [02-mapserver-01]: ../img/02-mapserver-01.png
