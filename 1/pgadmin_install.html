![][01-pgadmin-06] Альтернативой работе с базой данных через консоль является более интуитивно понятная среда клиента с графическим интерфейском, например pgAdmin. pgAdmin - это свободное кроссплатформенное программное обеспечение, то есть способное функционировать как в *nix, так и в Windows. Опишем процесс установки для Windows. 1. Переходим на страницу загрузок [pgAdmin](http://www.pgadmin.org/download/windows.php) и скачиваем последнюю версию программы, в данном случае это будет [pgAdmin v1.14.3](http://ftp.postgresql.org/pub/pgadmin3/release/v1.14.3/win32/pgadmin3-1.14.3.zip). 2. Распаковываем архив и запускаем файл `pgadmin3.msi`. 3. По окончанию установки запускаем программу: `Пуск -> Все программы -> pgAdmin III 1.16 -> pgAdmin III`. 4. В открывшемся окне выбираем `Файл -> Добавить сервер`. И указываем следующие параметры: * **Имя** - произвольная строка по которой вы сможете идентифицировать сервер в списке доступных серверов; * **Хост** - адрес хоста на котором запущен PostgreSQL; * **Порт** - порт, который прослушивает PostgfeSQL (по умолчанию 5432); * **Имя пользователя** - `pguser`; * **Пароль** - `topsecret`; ![][01-pgadmin-01] 5. В окне *Браузер объектов* выберите базу данных `dbvega`. ![][01-pgadmin-02] 6. Нажмите кнопку ![](../img/01-pgadmin-03.png) на панели инструментов для запуска редактора SQL. 7. В открывшемся окне наберите команду: SELECT postgis_full_version(); 8. Нажмите кнопку ![](../img/01-pgadmin-04.png), в результате чего будет выполнен запрос. На панели вывода результатов (расположенной снизу от окна редактора SQL) должен появиться результат выполнения команды аналогичный тому, что был получен на предыдущем этапе: ![][01-pgadmin-05] [01-pgadmin-01]: ../img/01-pgadmin-01.png [01-pgadmin-02]: ../img/01-pgadmin-02.png [01-pgadmin-05]: ../img/01-pgadmin-05.png [01-pgadmin-06]: ../img/01-pgadmin-06.png
